### Usage

Using the Flex component - Defaults to horizontal

```jsx
<Flex className="p-2">
  <div className="bg-gray-200 rounded-md p-2">One</div>
  <div className="bg-gray-200 rounded-md p-2">One</div>
  <div className="bg-gray-200 rounded-md p-2">One</div>
</Flex>
```

There is also Horizontal and Vertical wrappers to save you some typing

```jsx
<Vertical className="p-2">
  <div className="bg-gray-200 rounded-md p-2">One</div>
  <div className="bg-gray-200 rounded-md p-2">One</div>
  <div className="bg-gray-200 rounded-md p-2">One</div>
</Vertical>

<Horizontal className="p-2">
  <div className="bg-gray-200 rounded-md p-2">One</div>
  <div className="bg-gray-200 rounded-md p-2">One</div>
  <div className="bg-gray-200 rounded-md p-2">One</div>
</Horizontal>
```
